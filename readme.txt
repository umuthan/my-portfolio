== Description ==

Wordpress Portfolio Module

A Wordpress plugin to create your works in categories

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/my-portfolio` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create your works from admin panel

== Frequently Asked Questions ==

= What is portfolio? =

Portfolio is a custom post type to create your works and shows them on page.
